Support & Docs

Help for running NeuroStudy

🚀 Quick Start

1

Install dependencies: pip install -r requirements.txt

2

Set your API keys in agent.py and tools.py

3

Start backend: python main.py

4

Open index.html in browser and start researching!

💡 What to say to judges about MCP

"Our system implements the Model Context Protocol pattern — instead of hardcoding which tools to call, we register tool schemas with the LLM and let it autonomously decide at runtime which tool to invoke based on context. The LLM acts as the orchestrator, not us."

FAQ

Backend not connecting?

Make sure python main.py is running and update API_BASE in this file to your server URL.

Slow responses?

Switch to Quick Revision mode — it uses fewer tokens and runs faster for demos.

Deploying to Render?

Set API_BASE to your Render URL (e.g. https://your-app.onrender.com).
